# Expense Tracker CLI

A simple command-line expense tracking application written in Python.  
Users can record daily expenses including date, category, amount, and description.  
Data is stored persistently in a CSV file.

## Features
- Add new expenses
- CSV-based data storage
- Input validation for numeric values
- Beginner-friendly structure

## Technologies Used
- Python 3
- CSV module (standard library)
